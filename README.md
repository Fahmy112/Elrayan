# Elrayan Auto Service - Frontend

واجهة مركز الريان لإدارة الصيانة والمخزون (React + Material UI)

## التشغيل المحلي

```bash
npm install
npm start
```

## متغيرات البيئة

- `REACT_APP_API_URL` : رابط الـ backend (مثال: http://localhost:5000/api)
